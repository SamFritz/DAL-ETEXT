During the course of my time at the School of Information Management at Dalhousie University, I participated in INFO 6620: Elextronic Text Design. Here are some of the projects that came out of that course, demonstrating basic concepts of HTML and CSS design.

  • Previous INFO 6620 syllabus available.


Typography HTML Essay - designed to demonstrate xhtml and CSS skills and the ability to create a functional attractive, and accesible design for the content of your essay.
Alice Exerpt TEI XML Project - this project was designed to give participants experience in the application of the TEI XML encoding. The requirement of this project was to encode a document which was provided with TEI XML tags to ain an understanding of the principles and goals of text encoding and specifically of text encoding for the humanities.
Datatbase Book List - the goal was to create a database type document, where data is complex enough to offer an opportunity for different display options and opportunies.
